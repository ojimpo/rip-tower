# Rip Tower — 使い方ガイド

Discord で OpenClaw に話しかけて CD リッピングを操作する。

## 基本の流れ

1. CD をドライブに入れる
2. OpenClaw に「CD入れた」「リッピングして」と言う
3. 完了したら Discord に通知が来る
4. メタデータに問題があれば WebUI（http://rip-tower.arigato-nas/）で確認・編集

## よく使うフレーズ

### リッピング開始
- 「sr0 にCD入れた、リッピングして」
- 「白ロジテックでリップ、図書館CDで」
- 「3枚組の Disc 2、黒ロジテック1 で」

### 状態確認
- 「リッピングどうなってる？」
- 「ドライブの状態見せて」

### イジェクト
- 「sr0 イジェクトして」
- 「白ロジテック開けて」

### 承認
- 「レビュー待ちのやつ承認して」

### CD情報の確認（リッピング前）
- 「入れたCD何か調べて」

## ソースタイプ

リッピング開始時に伝えると分類される：
- **図書館** — kashidashi と連携して貸出管理に反映
- **手持ち** — 自分のCD
- 何も言わなければ「未分類」

## 複数枚組

- 「2枚組の Disc 1」と伝える
- OpenClaw が disc_number と total_discs を設定してリッピング開始
- Disc 2 を入れたら「Disc 2 リッピングして」

## 細かい操作は WebUI で

以下は WebUI（http://rip-tower.arigato-nas/）から：
- メタデータの手動編集（アーティスト名、トラックタイトル等）
- アートワークの選択・アップロード
- 歌詞の確認・編集
- kashidashi マッチの手動選択
- 設定変更（エンコード形式、テンプレート等）
- WAV インポート（別マシンでリップしたファイル）
